# DetectImageBlur

Java project (Spring Boot) that exposes a web service to determine if an image is blurred or not.

This application is based on the OpenVC library.

Documentation:
Installing OpenCV for Java, Link: https://opencv-java-tutorials.readthedocs.io/en/latest/01-installing-opencv-for-java.html
Blur detection with OpenCV, Link: https://www.pyimagesearch.com/2015/09/07/blur-detection-with-opencv/
ImageBlur Detection, Link: https://github.com/jainullas/ImageBlurDetection
